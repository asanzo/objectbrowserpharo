Requiere de una instalacion aparte. Puede obtenerse la ultima version de la pagina oficial del proyecto Graphviz: http://www.graphviz.org/Download..phpAsimismo con esta distribucion se puede encontrar una version para cada plataforma dependiendo del sistema operativo:	- Windows	- Unix/Linux	- MacOS------------------------------------------------------------------------------------------------------------------------------------------------------------------Para tener el sistema Loop funcionando se debe:NOTA: Si ya se encuentra instalado el entorno Pharo en su sistema puede saltearse al unto 4 directamente1) OPCIONAL: Instalar Graphviz2) Instalar la VM-Pharo3) Descomprimir el archivo: PharoV10.sources en donde se haya instalado la VM4) Descomprimir el archivo Loop en donde se haya instalado la VM5) Arrastrar el archivo Loop.image a la VM instalada6) Empezar a usar LOOP7) Configuracion del graficador:	7.1) Si se desea usar Graphviz no se requieren mas pasos pues es el graficador por default. 	7.2) Si se desea usar SmallUML, se debe abrir un workspace y ejecutar el siguiente comando: "LOOPObjectDiagramFactory diagramClass: SmallUmlObjectDiagram. "	------------------------------------------------------------------------------------------------------------------------------------------------------------------En la carpeta Documentacion encontraras el manual de usuario del entorno Loop y un documento explicativo de la arquitectura del sistema si queres interiorizarte sobre como fue desarrollado Loop.----------------------------------------------------------------------------------------------------------------------------------------------------------------....@@@@@@........................................@@@@..............................@@@@..@@@@......................................@@@@............................@@@@......@@@@....@@@@@@@@......@@@@@@......@@@@@@@@@@....@@@@@@@@......@@@@@@@@..@@@@......@@@@..@@@@....@@@@..@@....@@@@..@@@@....@@@@..@@@@....@@@@..@@@@....@@@@@@@@......@@@@..@@@@@@@@........@@@@@@@@..@@@@....@@@@..@@@@....@@@@..@@@@@@@@....@@@@......@@@@......@@@@@@@@..@@@@..@@@@..@@@@....@@@@..@@@@....@@@@......@@@@@@@@..@@@@..@@@@....@@@@....@@@@..@@@@..@@@@..@@@@....@@@@..@@@@....@@@@..@@@@....@@@@....@@@@@@........@@@@@@@@......@@@@@@@@....@@@@@@@@@@....@@@@@@@@......@@@@@@@@..----------------------------------------------------------------------------------------------------------------------------------------------------------------Para tener la ultima version (de desarrollo).1) Abrir Monticello y agregar un nuevo repositorio: /ObjectBrowser2) Ecribir en un workspace:	Gofer it    squeaksource: 'ObjectBrowser';    package: 'ConfigurationOfObjectBrowser';    load.	(Smalltalk at: #ConfigurationOfObjectBrowser) project latestVersion load.3) Evaluarlo
